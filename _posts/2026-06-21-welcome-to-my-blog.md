## 안녕하세요 👋

GitHub Pages와 Jekyll(Chirpy 테마)로 만든 첫 글입니다.
앞으로 공부한 내용과 개발 기록을 차곡차곡 남겨보려고 합니다.

## 글은 이렇게 작성해요

`_posts` 폴더에 `YYYY-MM-DD-제목.md` 형식으로 파일을 만들면 됩니다.
파일 맨 위 `---` 사이의 영역(front matter)에 제목·날짜·카테고리·태그를 적어요.

### 코드 블록 예시

```python
def hello():
    print("Hello, blog!")
```

### 인용 / 강조

> 꾸준함이 가장 강력합니다.
{: .prompt-tip }

이제 이 파일을 지우거나 수정해서 나만의 첫 글을 써보세요!
